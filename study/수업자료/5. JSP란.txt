JSP
 - Java Server Page의 줄임말
 - 서블릿의 디자인 버전
 - 서블릿의 상당부분을 생략해주고 나중에 자동으로 변환(jsp -> servlet)
 - 주소가 자동으로 부여(파일명)

구성요소
[1] 디렉티브(directive)
 - 페이지의 설정을 담당하는 영역		<%@ 코드 %>
[2] 스크립트릿(scriptlet)
 - 실행될 자바 코드를 설정하는 영역	<% 코드 %>
 - 자동으로 변환 시 service()에 포함되는 코드
 - 서블릿에서 작성하던 코드를 작성할 수 있음
[3] 표현식(expression)		<%=값%>
 - out.println()을 축약한 표현
 - 값을 출력할 때 유용하게 사용
[4] 이 외의 모든 값들은 글자로 인식
